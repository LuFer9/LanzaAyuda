BASE DE DATOS AppHotel


Tenemos una base de datos propia en la cual cuando el cliente registra sus datos junto con una reserva tato de salón como de habitación, esta se actualiza con dichos datos.


Tenemos distintas tablas como la tabla cliente, en la cual se almacenan los datos de personales de cada cliente como el nombre, DNI etc.

La tabla provincias donde se almacena el lugar de residencia de los clientes

La tabla reservas de Salón donde se guardan los datos relacionados a la reserva de un Salón

Y por ultimo la tabla de reservas de Habitaciones donde se guardan los datos relacionados con las reservas de habitaciones
